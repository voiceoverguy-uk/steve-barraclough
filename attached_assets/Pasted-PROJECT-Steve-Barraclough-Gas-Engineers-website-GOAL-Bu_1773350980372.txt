PROJECT: Steve Barraclough Gas Engineers website
GOAL: Build a clean, modern, mobile-friendly single-page website for a local father-and-son gas engineering business based in Pontefract, covering West Yorkshire. The site should feel trustworthy, established, credible, and affordable — not cheap-looking, but also not too premium or corporate. It should be built in a format suitable for deployment on Vercel using Next.js.

TECH STACK:
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Mobile-first responsive design
- Ready for Vercel deployment
- Structure code cleanly so a Resend contact form can be added later

BRAND / DESIGN DIRECTION:
- Use a modern, polished trades-business style
- Tone: professional, friendly, reliable, local, father-and-son, experienced
- Avoid anything overly flashy, gimmicky, or too “agency”
- Colour palette:
  - Dark navy / near-black for hero background overlays
  - White for text
  - Worcester Bosch blue as a main accent colour
  - Gas Safe yellow as a secondary accent colour
- Use the yellow subtly for hover underline animations, small highlights, badges, or dividers
- Use a clean sans-serif typeface with a Century Gothic-like feel
- Plenty of spacing, rounded cards, subtle shadows, modern buttons
- Make the whole site look far better than typical outdated local gas engineer websites
